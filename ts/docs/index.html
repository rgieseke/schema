@stencila/schema

Stencila Schema for Typescript

This package provides Typescript bindings for the Stencila Schema.It is primarily aimed at Typescript and Javascript developers wanting to programmatically generate, or modify, executable documents. For example, it is used in encoda, a library for converting between document formats, and in basha, an interpreter for executable documents containing Bash code.

Install

 
npm install @stencila/schema --save
